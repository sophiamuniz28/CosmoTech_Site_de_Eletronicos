# 🚀 CosmoTech - Loja Virtual de Eletrônicos

Projeto de site de vendas online desenvolvido com HTML e CSS, com foco em layout moderno, organização e experiência do usuário.

---

## 🖥️ Sobre o Projeto

A **CosmoTech** é uma loja virtual fictícia criada para aplicar na prática conceitos de desenvolvimento web.

O projeto simula um e-commerce de eletrônicos, com navegação entre páginas, exibição de produtos e formulário de contato.

---

## ✨ Funcionalidades

* 🛒 Página inicial com apresentação da loja
* 📦 Página de produtos com layout em Grid
* 📞 Página de contato com formulário
* 🎨 Estilização completa com CSS
* 💡 Efeitos de interação (hover e animações)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento front-end, focando em:

* Estruturação de páginas com HTML
* Estilização com CSS
* Uso de Grid e Flexbox
* Criação de interfaces organizadas e responsivas
